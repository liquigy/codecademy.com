How to use

This website contains a variety of selectors assigned to a specific color of background, text, border, font weight and size. There are also selectors to apply the specific font for any tag in HTML. Simply find the tag in the HTML and then add or replace the selector with the one that you desire.

FONT EXAMPLE <span class="abril font-size-30 font-weight-400">

OUTPUT This is the example of Abril Fatface font, size 30px, with 700 boldness

COLOR EXAMPLE <span class="clryellow bgred borderblack">

OUTPUT This is the example of yellow text, red background, black border

There is a template broken down below which you can yourself modify using the Website Builder guide.

If you wish to use the selectors on any other HTML be sure to link the font.css and colors.css in the head of your own HTML.

Have fun and enjoy coding!
