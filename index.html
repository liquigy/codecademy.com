How to use

This website contains a variety of selectors assigned to a specific color of background, text, border, as well as font type, weight, size, style and alignment. You can add or replace any of the selector(s) below to any tag in HTML that you like.

FONT EXAMPLE <span class="abril font-size-30 font-weight-400">

OUTPUT This is the example of Abril Fatface font, size 30px, with 700 font weight

COLOR EXAMPLE <span class="clryellow bgred borderblack">

OUTPUT This is the example of yellow text, red background, black border

Check out the TEMPLATE before you see the breakdown of the selector uses at the bottom of this page.

If you wish to use the selectors of the CSS on any other HTML page be sure to copy the font.css, colors.css, and otherselectors.css in the folder with your own web page and don't forget to link them in the head of your HTML.

Have fun and enjoy coding!
